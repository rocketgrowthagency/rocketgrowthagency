# Rocket Growth Agency — Vite + React + Tailwind

## Local dev
1) `npm install`
2) `npm run dev`

## Build
`npm run build` → outputs to `dist/`

## Netlify
- Build command: `npm run build`
- Publish directory: `dist`
- Forms: The audit form is Netlify Forms‑ready (`data-netlify="true"`). Enable notifications in Netlify UI.
